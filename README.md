# Catch the Falling Fruits 🍎

This is a simple **two-player console game** developed in **C++** as my **Programming Fundamentals (PF)** semester project. The objective of the game is to catch falling fruits to increase your score while avoiding bombs that reduce it.

This project helped me strengthen my understanding of fundamental programming concepts by applying them in a fun and interactive game.

## Features

* Two-player gameplay
* Randomly falling fruits and bombs
* Live score tracking for both players
* Winner announcement at the end of the game
* Keyboard controls for both players
* Option to quit the game at any time

## Controls

### Player 1

* **A** – Move Left
* **D** – Move Right

### Player 2

* **J** – Move Left
* **L** – Move Right

Press **Q** to quit the game.

## Scoring

* Catching a **Fruit (O)** increases your score by **1**.
* Catching a **Bomb (X)** decreases your score by **1**.

## Concepts Used

While developing this project, I practiced several Programming Fundamentals concepts, including:

* Functions
* Structures (`struct`)
* Arrays
* Loops
* Conditional statements
* Random number generation
* Console input/output
* Basic game logic

## How to Run

1. Open `Catch_The_Falling_Fruit.cpp` in Code::Blocks or any C++ IDE.
2. Compile the program.
3. Run the executable.
4. Follow the on-screen instructions and enjoy the game.
